TrucCompression

Lossless experimental Math Formula Codec (MFC1) — a block instruction compressor that picks the cheapest reversible transform per block: CONST, REPEAT, COPY, ZLIB, BZ2, LZMA, DELTA+ZLIB, XOR+ZLIB, or RAW.

GitHub Quick start Bounties

Why MFC?

Classic codecs apply one algorithm to the whole stream. MFC frames the file as a list of formulas (block ops). Each block is scored and the smallest reversible payload wins. Round-trip is always byte-for-byte verified with SHA-256 in the container header.

Block operations

CONST / REPEAT

Uniform bytes and tiled patterns store as tiny payloads.

COPY_PREVIOUS

Identical consecutive blocks store as a reference only.

ZLIB · BZ2 · LZMA

General entropy coding candidates per block.

DELTA / XOR + ZLIB

Predictive transforms for ramps and correlated streams.

Install & demo

git clone https://github.com/mergeos-bounties/TrucCompression.git
cd TrucCompression
python -m venv .venv
# Windows: .venv\Scripts\activate
pip install -e ".[dev]"
truccompression version
truccompression demo
truccompression compress input.bin out.mfc --verify
truccompression decompress out.mfc restored.bin

Honest limits

  • Already-compressed / high-entropy data may grow (RAW fallback).
  • Experimental format — not a replacement for zip/zstd in production yet.
  • No secrets in fixtures; offline demo needs no network.
