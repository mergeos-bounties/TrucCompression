# TrucCompression bounties

Open issues labeled `bounty` on this repo. Typical scale: **25 / 50 / 100 / 200 MRG**.

## Claim

1. Follow org + star `mergeos` and `mergeos-contracts`.
2. Comment on the issue to claim.
3. PR to **master** with tests green.
4. Maintainer merges → ledger credit + comment.

## Good first areas

- New reversible block ops (with uniqueness tests)
- Better pattern detectors (RLE, BWT variants)
- CLI UX (`bench` formats, progress)
- Web demo enhancements on GitHub Pages
- Corpus fixtures under `data/samples/`
